# marombaAda

Bem-vindo ao **marombaAda**, um sistema simples de gestão de dietas e treinos desenvolvido como exercício para treinar conceitos básicos de POO. Este projeto foi criado para ajudar atletas a gerenciar suas rotinas de treino e alimentação de maneira eficiente e personalizada.

## Índice

- [Descrição do Projeto](#descrição-do-projeto)
- [Funcionalidades](#funcionalidades)

## Descrição do Projeto

O **marombaAda** é uma aplicação que permite a gestão de rotinas de treinos e dietas para atletas de diversas modalidades. Seja você um nadador, praticante de musculação ou de qualquer outra atividade física, o sistema é projetado para suportar suas necessidades específicas, garantindo que sua alimentação e treinos sejam acompanhados de organizada.

## Funcionalidades

- **Gestão de Treinos**: Crie e personalize planos de treino para diversas modalidades esportivas, como natação, musculação, corrida, entre outras.
- **Gestão de Dietas**: Registre e acompanhe dietas personalizadas, incluindo tipos variados de alimentos, quantidades e horários das refeições.
- **Suporte a Múltiplas Modalidades**: Adapte o sistema para diferentes tipos de atletas e modalidades esportivas, garantindo flexibilidade e personalização.
